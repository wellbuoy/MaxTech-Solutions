# MaxTech Solutions-SA

A modern, responsive landing page for MaxTech Solutions, a technology provider specializing in computer repairs, accessory sales, and IT support.

## Overview

This project is a static website designed to showcase services and products. It features a clean, modern interface with smooth scrolling, interactive elements, and a responsive layout that works across devices.

## Features

- **Responsive Design**: Adapts seamlessly to mobile, tablet, and desktop screens.
- **Service Showcase**: persistent grid layout displaying core services.
- **Product Gallery**: Interactive product grid with detailed modal views.
- **Interactive UI**: Custom styling with hover effects, smooth transitions, and a mobile-friendly navigation menu.
- **FAQ Section**: Accordion-style Frequently Asked Questions.
- **Contact Info**: integrated contact section.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, Grid, and Media Queries for styling and responsivness.
- **JavaScript**: Vanilla JS for mobile menu toggling, product modals, and potential interactions.

## Setup & Usage

Since this is a static site, no build process is required.

1.  Clone the repository or download the files.
2.  Open `index.html` in your preferred web browser.

## File Structure

- `index.html`: Main HTML file containing structure and inline styles/scripts.
- `*.jpg/png`: Assets for products and layout.
